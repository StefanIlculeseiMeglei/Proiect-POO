Programul este X si 0 vs calculator si este o aplicatie pentru platforma Linux.
Ca aplicația să funcționeze trebuie sa fie instalat compilatorul gcc si makefile.
Pentru a instala make scrieti aceasta comanda in terminal: sudo apt install make
Pentru a instala compilatorul gcc scrieti aceasta comanda in terminal:  sudo apt install build-essential
Librăriile folosite sunt: termios.h, iostream, fstream.
Aplicația funcționează doar dacă fișierul info.txt există, fișierele save.txt și puncte.txt se creează automat.
